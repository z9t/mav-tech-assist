This folder is copied into the USB bundle.

Bundled by default:
- iperf3.exe
- cygcrypto-3.dll
- cygwin1.dll
- cygz.dll

Optional extras:
- speedtest.exe

The scripts look here first before falling back to PATH.

If you refresh the bundled iperf3 copy, keep the DLL files with it.
See iperf3-source.txt for the source URL and checksums.

For the configured throughput test, run an iperf3 server on:
showtime.mav.z9t.me:5201
